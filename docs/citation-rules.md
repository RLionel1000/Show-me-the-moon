# Citation and Attribution Rules

This project is MIT licensed, but attribution still matters for ethics, clarity, and third-party compliance.

## Project-level rule

If you redistribute Show me the moon or a derivative:

- keep `LICENSE`
- keep third-party notices
- credit Show me the moon in your README or equivalent public project page
- summarize your changes

## Recommended attribution block

```md
## Attribution
This project is based on Show me the moon, licensed under MIT.
Changes made in this fork: <short summary>.
```

## Data and media rule

If you keep bundled third-party media or data:

- preserve relevant source and license context
- do not strip attribution notes from repository documentation
- update attribution docs when assets change

Relevant local files include:

- `docs/planet-image-attribution.md`
- `assets/moon/README.md`
- `README.md`

## Template ecosystem rule

If you publish a custom template based on Show me the moon:

- credit Show me the moon as the base platform
- make clear which part is your template-specific work
- keep any third-party asset attribution required by your template
